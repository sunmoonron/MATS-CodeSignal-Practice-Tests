# List of Toy Tasks 
- **Booking**: Event booking system with capacity management and waitlists.
- **File Sync**: Directory synchronization tool that computes diffs and syncs files.
- **KVStore**: In-memory key-value store with TTL expiration, compare-and-set operations, and JSON export/import.
- **Log Analyzer**: Log parsing and analysis tool.
- **Markdown to HTML**: Markdown to HTML converter.
- **Mini Grep**: Simple grep utility with various options.
- **Simple HTTP Server**: Basic HTTP server with JSON API for items.
- **Todo**: Task manager with tags, due dates, and filtering.
- **URL Shortener**: URL shortening service with custom aliases and click tracking.
